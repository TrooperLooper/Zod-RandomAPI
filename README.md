# Zod Data Validation & Express Backend

A TypeScript backend server demonstrating schema validation, API integration, and production-grade error handling. This project showcases proficiency in Express.js, Zod runtime validation, asynchronous programming, and RESTful API design. The application fetches data from randomuser.me, validates structured data with Zod schemas, and returns properly typed responses with comprehensive error handling using try-catch blocks and appropriate HTTP status codes. 
<br><br>
Technologies: Express.js 5.x, TypeScript 5.x, Zod 4.x, ts-node, nodemon.
